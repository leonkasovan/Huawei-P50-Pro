```
Controller Setting -> Port 1

D-Pad Up : -Axis1
D-Pad Right : +Axis0
D-Pad Down : +Axis1
D-Pad Left : -Axis0
Triangle: Button100
Circle: Button97
Cross: Button96
Square: Button99

AppSettings -> System
Enable Rewinding: on

AppSettings -> Graphics
GPU Renderer: Hardware (Vulcan)
Resulution Scale: 4x
Texture Filtering: Bilinear
```
